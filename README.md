jquery.tabs.js
==============
Lightweight, Provide a tab function with jQuery

## Demo
### [See Basic Demo](https://shoyan.github.io/jquery.tabs.js/)

## Usage
```
$(function(){
  $(".js-tabs").tabs({active_tab: "#tab1"});
});
```
